## Project Description

ISKO-ON-THE-GO is a dedicated mobile application designed to help students of the Philippine Science High School - CALABARZON Region Campus. It is an all-in-one platform where you can access real-time announcements, class schedules, the list for Lost
and Found items, the student handbook details, etc.

With requirements piling up on each other, ISKO-ON-THE-GO addresses this challenge as a unified platform where scholars have complete and quick access to all their school related information. With this in mind, we aim to streamline the student experience for both the upperclassmen and the incoming freshmen.

Our app makes life easier by saving the time you would usually take to look for upcoming assignments and announcements by sending them straight to you through the app's announcements. Moreover, our app also saves a lot on phone storage as you wouldn't need multiple apps for any of your school needs as they are all here. This makes our app a very effective way to save time and space on any of the requirements and important announcements as well as being valuable for academic research, technical documentation, and various assignments that demand a unified framework.

## Objectives

-Implement a secure authentication system that validates users via the institutional email given by the school and restricts access to any invalid inputs.

-Create an announcement dashboard capable of filtering and displaying messages based off the user's grade, class, and batch.

-Integrate a scheduling module that displays the residence hall routines for interns.

-Create a digital Student ID profile that stores personal info, SCALE/ALP details, and allows for user-driven data updates.

-Design a searchable campus database for lists for borrowed library books and the Student Council Organization (SCO) Lost and Found registry.

-Create a digital Student Handbook module that displays the campus dress code, uniform types, and official rules and regulations for quick reference.

-Establish a functional "Log Out" and Exit system so that the user can leave at any time.

## Core Features

• Secure Student Authentication & Validation
     - Implements a login system that requires school-provided institutional email addresses to ensure that only verified campus members can access sensitive academic data.
  
    - Features automated validation system that cross-references user-inputted grade levels and sections, preventing data conflicts and ensuring students only see content relevant to their specific academic standing.
  
●  Announcement Dashboard
    - A provided announcement dashboard which shows only the relevant  announcements based off their grade level and section.
  
● ‌Class Schedules
    - User will be provided a digital interface for students to view their specific class          schedules per grade and section. This eliminates the need for physical copies and ensures scholars are always aware of their current period's location and time.

● ‌Integrated Student Council Office Lost & Found 
    - A digital version of the SCO’s "Lost and Found" list where students can check for any of their missing belongings in real time, reducing the need to visit the SCO office physically and making the recovery process for lost items much easier.

● ‌Session Management
    - Includes a functional "Log Out" and "Exit" system that shuts off all previous actions to   ensure that student information remains private and secure when the device is not in use.

## Additional Features
● ‌Digital Student Handbook & Dress Code Guide
    - Provides a portable, searchable version of the campus rules and regulations.
    - Includes detailed sections on the campus dress code and specific uniform types required for different school activities or specific days of the week, helping scholars remain compliant with campus standards.

## How to Run the Program
● Step-by-step instructions
- 1. Make sure to have Dev C++ or a compiler installed on your device.
- 2. Download the ISKO_ON_THE_GO.cpp file (should come from a current
  student/member of the Pisay CBZRC community).
- 3. Open your file manager app.
- 4. Search for ISKO_ON_THE_GO.cpp or see it in quick access/recents.
- 5. Click the file to launch it (Double-click on desktop/laptop).
- 6. Use the program as needed.
- 7. Refer to the README file for any concerns.
